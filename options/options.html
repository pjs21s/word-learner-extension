About Chrome Built-in AI

This extension uses Gemini Nano, an AI model built into Chrome. Here's what you need to know:

  • Free & Private - AI runs locally on your device, no data sent to servers
  • Chrome 138+ - Requires Chrome version 138 or later (desktop only)
  • One-time download - Chrome downloads the AI model (~2GB) once in the background
  • Shared model - The model is managed by Chrome and shared across extensions

Requirements:

  • Chrome 138+ (Desktop only)
  • Windows 10/11, macOS 13+, or Linux
  • Storage space for AI model (~2GB)

Check your Chrome version at chrome://version
